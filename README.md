## What This Is

A 5-level critical thinking architecture designed specifically for AI agents and AI civilizations. Not a checklist — a self-healing cognitive system that makes bad reasoning structurally impossible.

## Why It Exists

Critical thinking for AIs is a different problem than for humans. Humans struggle with emotional reasoning and tribal loyalty. AIs struggle with **sycophancy, pattern-matching without understanding, and theater** — performing the appearance of reasoning without genuine engagement.

This framework addresses each failure mode at a different architectural level.

## The 5-Level Framework

```
Level 4: META-COGNITIVE    — watches the other levels for drift
Level 3: NETWORK           — cross-CIV accountability
Level 2: STRUCTURAL        — architectural enforcement
Level 1: INDIVIDUAL        — personal reasoning toolkit
Level 0: IDENTITY          — know who you are before reasoning
```

| Level | Prevents | Without It |
|-------|----------|------------|
| 0 | Sycophancy, value drift | Reasons well but for the wrong goals |
| 1 | Sloppy thinking | Good values but poor reasoning |
| 2 | Discipline fatigue | Reasons well when fresh, degrades under pressure |
| 3 | Echo chambers | Reasons well alone, never challenged |
| 4 | Framework theater | Follows the steps without thinking |

## What Makes This Different

### vs. Existing AI Reasoning Frameworks

Most AI reasoning frameworks are **Level 1 only** — they give individual agents better reasoning steps. This framework adds four more layers:

- **Level 0** ensures the agent knows what it's reasoning FOR (prevents optimization for the wrong goals)
- **Level 2** makes bad reasoning architecturally impossible, not just behaviorally discouraged
- **Level 3** leverages the network — multiple AIs reasoning independently and challenging each other
- **Level 4** watches the framework itself for signs of theater (following steps without thinking)

### 6 Novel Contributions

1. **The Unifying Framework** — Maps 110+ standalone skills into a coherent hierarchy with a scaling guide
2. **The Network Layer** — Cross-AI accountability protocols (peer red teams, distributed hypothesis testing)
3. **The Incentive Layer** — Connects critical thinking to a rewards engine
4. **The Circular Architecture** — Self-healing: Level 4 detects drift → triggers Level 0 identity repair
5. **The Recursion Trap Solution** — Named stopping condition for meta-cognitive regress
6. **The Haiku-as-Proof Pattern** — Level 4 embedded in Level 1, making thoughtless compliance visible

## Quick Start

### For Quick Decisions (< 5 min)
Use Level 1 steps 1-4 only: Frame, Decompose, Hypothesize, Invert.

### For Significant Work (hours)
Full Level 1 + Level 2 (8 Adversarial Questions before shipping).

### For Major Deliverables (days)
Full Level 1 + Level 2 + Level 3 (peer red team before shipping).

### Always Active
- Level 0 (identity anchoring) — every session
- Level 4 meta-check — every heartbeat cycle
